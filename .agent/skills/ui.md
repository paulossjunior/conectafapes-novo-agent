# Skill: UI (Nuxt UI optional + Common Components)

## Primary UI
- Se Nuxt UI estiver habilitado, preferir componentes Nuxt UI
- Se não, preferir `src/common/components/*`

## UX rules
- Sempre ter loading / empty / error
- Feedback padrão: snackbar/toast/alert
- Acessibilidade básica (labels, foco, teclado)
