# LEDS .agent (Full)

Este pacote define um agente especialista em **LEDS Modular Frontend** (Vue 3 + TS + Vite + Vue Router federado + Pinia) e integração HTTP via **Axios encapsulado** em `src/common/api`.

## Estrutura
- `persona.md` — identidade e regras do agente
- `skills/` — competências e padrões (arquitetura, Vue, API, state, UI, forms, tables)
- `memory/` — contexto do projeto (stack, boundaries, naming, UI patterns)
- `workflows/` — passo a passo de criação de módulos/features/CRUD e revisão de PR
- `skills/snippets/` — exemplos prontos para copiar/colar
